# Detective Jr.

A single-file HTML detective game for kids (ages 8–10), built on Sherlock Holmes'
*A Scandal in Bohemia*. Sweep a torch over the dark scene to reveal the clue, then
tap what you found.

**▶ Play:** https://mikekhanhle-droid.github.io/detective-jr/

Published from the private development repo. The whole game is `index.html` plus the
images in `assets/` — no build step, no server needed.
